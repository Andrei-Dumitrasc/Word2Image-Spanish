Please upload the file containing the plain, unformatted text to be used as input. The file should be saved in the .txt format.


To this end, put your text in Notepad, then choose File -> Save As...
There, give a name to your file and make sure the type is Text Documents (*.txt). Save this file in an easily accesible folder.
Then, click the Browse button on this page to choose this file and then on the Upload text button.
