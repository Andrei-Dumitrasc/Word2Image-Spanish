Word2Image

The form below will help you define which words are to be replaced, and by which image.

Please fill the field below with one valid word (i.e. no numbers, symbols etc), then click on the button Confirm word.
Related forms (e.g. plural) will be then automatically generated. From this list select those that you wish to replace. To select more than one, hold the CTRL key down until you are done with clicking on the words.
Then, to the right, upload the image by which you wish to replace the given words.


You can get images from the Internet by right-clicking on them and choosing the option Save Image As...
There, you can choose a name to help you remember this image. If the file type is .gif please use a different image. Save this file in an easily accesible folder.
Then, click the Browse button on this page to choose this file and then on the Upload image and confirm words button.
